# Tools

Personal utility scripts and tools, organized by category.

## Categories

| Directory | Description |
|-----------|-------------|
| [backup/](backup/) | Backup and restore utilities |

## Structure

Each category is a subdirectory containing one or more tool folders. Every tool folder includes its own README with setup and usage instructions.

```
tools/
└── backup/
    └── restic_bash/   # Automatic WSL restic backup
```
